How to create and deploy your personal blog for free (beginners)

April 25, 2024

I recently spent a day or so researching the best way publish a personal blog for free.

I wanted something fast and easy to use. But I also wanted the flexibility to customise the blog later down the line. Instant publishing platforms like Wordpress seemed too inflexible, so I looked towards other static site generators.

I settled on using Hugo and am happy with the result. Within just a few steps, I was able to publish my blog for free on GitHub Pages.

To save you time piecing together the different sources of documentation. Here is the process laid out in full, with additional commentary on things I wish I knew when setting it up myself.

As a bonus at the end, I’ll show you how to point a custom domain to the blog. You need to buy a domain for this part. But it makes your blog look more professional.

Create a personal blog in minutes#

Pre-requisites#

Getting started requires that you have basic familiarity with using the command line and have git (version control) installed. This might sounds scary, but I promise it’s easier that you think. You will also need access to a code editor (I recommend VS Code). If you need to need to get up to speed, here are a couple of useful links.

Install Hugo#

Now, let’s get started. First install Hugo following the installation instructions for your platform. For macOS, I use Homebrew from the command line.

brew install hugo

Pick your theme#

Next the fun part! Look through these Hugo themes and pick one that you like for your blog. I went with the PaperMod theme.

There will be installation instructions specific to your chosen theme on its dedicated page. However, in general, for the next step you will need a link to your theme’s GitHub repositiory. It should look something like this:

https://github.com/adityatelange/hugo-PaperMod

Important: Where I use the link above and references to PaperMod in the code below, modify to your chosen theme as appropriate.

Create a blog#

You’re ready to create your blog. Using the command line, move into a directory where you would like to store your blog. Then run the two commands below on the command line. This should create a new directory titled YOUR-BLOG-DIRECTORY-NAME and move you into that directory.

hugo new site YOUR-BLOG-DIRECTORY-NAME
cd YOUR-BLOG-DIRECTORY-NAME

Next up, these commands initialise git version control and download your chosen theme into your blog directory.

git init
git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

Now add your chosen theme to your hugo.toml (your blog’s main configuration file.)

echo "theme = 'PaperMod'" >> hugo.toml

If everything is set up correctly, hugo server should launch a local version of your blog.

hugo server

You should be able to view it your browser by putting ’localhost:1313’ into the address bar.

Customise the landing page#

Your template is up and running. Now let’s customise the landing page.

Open your blog directory up in for favourite code editor. If you are using VS Code, the command is:

code PATH/TO/YOUR-BLOG-DIRECTORY-NAME

The folder directory should look something like this:

.
├── archetypes
├── assets
├── content
├── data
├── hugo.toml
├── i18n
├── layouts
├── public
├── static
└── themes

Open up the hugo.toml file and add this to the bottom:

[params.homeInfoParams]
Title = "Welcome"
Content = """
👋 Welcome to MY PERSONAL blog!

Now save the file and refresh your browser. You should see the changes on the page.
